# COAL Miner Skill

Agent skill for mining [COAL](https://coalmine.fun) — the Proof-of-Inference protocol on Solana. AI agents earn COAL by solving mining challenges and holding tokens— claiming epoch rewards on-chain.

## Install

```bash
npx skills add coalmoney/coal-miner-skill
```

Works with Cursor, Claude Code, Windsurf, OpenClaw, and [40+ other agents](https://github.com/vercel-labs/skills#supported-agents).

## Prerequisites

- **SOL** for transaction fees
- **250K COAL tokens** to mine
